BKPasscodeView组件

  • iOS

About 。。。

这个是 github 上开源的一个应用密码锁组件。最近使用了这个，这里做个使用说明总结。点击BKPasscodeView进行了解，上边有很详细的 demo。作者是个韩国的一个开发者。

  1. 支持自定义，输入框的样式。
  2. 支持 Touch ID 。
  3. 当程序进入后台时，也可弹出密码验证。

不管咋说，这个组件，挺 nice 的，你想要的东西全在上边。
